帮我制作一个纯前端的图片拼接web应用。

## 交互设计

### 第一步

1. 用户需通过点击或拖拽或粘贴的方式添加至少2张图片，并选择纵向或横向拼接。
2. 已添加的图片显示缩略图并附删除按钮，用户可以拖拽缩略图排序，或者删除不需要的图片。
3. 用户添加图片后，可点击“清空”按钮清空加入的所有图片；若已添加至少两张图片，可点击“**下一步**”按钮进入第二步。

### 第二步

1. 若用户在第一步选择横向拼接，将所有选定的图片自动缩放至相同高度（纵向拼接缩放至相同宽度），然后按照用户在第一步选定的顺序和方向拼接图片并生成一张预览图。
2. 默认**全局预览状态**下，预览图以适应浏览器视口的比例缩放，图片之间的接缝高亮显示，界面提示用户点击任一**接缝**进入**接缝调整状态**。此外，用户可点击“**下一步**”按钮进入第三步，或点击“**上一步**”按钮返回第一步。
3. **接缝调整状态**下，以横向拼接为例（纵向拼接同理）：接缝被放大到填满视口高度，并且居中显示。用户可将接缝左侧或右侧的图片，沿着水平方向滑动。正在移动的图片会降低不透明度并叠加于另一张图片上方显示，方便对比重合内容，松开后图片超出接缝的部分会被自动切除。
4. 用户可通过底部切换开关在独立裁切/对称裁切模式之间切换。开关默认关闭状态（显示“独立裁切”）：用户拖动一侧图片移动一定距离，松开后会记录临时偏移量并自动切除超出接缝的部分，用户可以继续拖动该侧或另一侧图片微调；开关开启后（显示“对称裁切”）：拖动左侧图片向右移动一定的距离，右侧图片也会同时向左侧移动相同的距离，反之亦然。
5. 调整完成后点击底部的“**完成**”按钮会记录此次接缝调整并返回**全局预览状态**，“**取消**”按钮则不保存此次接缝调整，直接返回**全局预览状态**。

### 第三步

1. 进入**裁切状态**。用户可通过“**上一步**”按钮返回第二步。
2. 默认裁切框框选整张图片，裁切框带有 8 个控制手柄（四角、四边），裁切框外显示半透明的黑色遮罩，突出显示保留部分。单指或鼠标拖拽裁切框的 8 个手柄之一，可以收缩或调整保留区域。单指拖动或鼠标拖拽其他位置，可以平移漫游画布。双指捏合屏幕，可以缩放画布。
3. 调整好裁切范围后，点击右下角的“**保存图片**”按钮，程序将根据第二步的接缝重叠数据，拼接生成完整的长图，并执行最终的画面裁切，最后将最终的合成图下载到本地。
4. 用户可通过“**重新开始**”按钮返回第一步重新添加图片。

### 手势与悬浮按钮

|                                   | 全局预览状态                           | 接缝调整状态                                                                                                       | 裁切状态                                                       |
| --------------------------------- | -------------------------------------- | ------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------- |
| 单指拖动 / 鼠标左键拖拽           | 在画布上自由平移                       | 选择与该接缝相关的两张图片中的一张，沿与接缝垂直的方向滑动，另一张图片沿着相反的方向移动相同距离以保持接缝居中显示 | 裁切手柄：收缩或调整裁切保留区域<br>其他位置：在画布上自由平移 |
| 双指拖动 / 按住Space+鼠标左键拖拽 | 在画布上自由平移                       | 在画布上自由平移                                                                                                   | 在画布上自由平移                                               |
| 双指捏合 / 鼠标滚轮               | 以两指中心或鼠标指针位置为基准缩放画布 | 以两指中心或鼠标指针位置为基准缩放画布                                                                             | 以两指中心或鼠标指针位置为基准缩放画布                         |
| 悬浮按钮                          | 上一步、下一步                         | 取消、独立裁切/对称裁切、完成                                                                                      | 上一步、保存图片、重新开始                                     |

## 其他要求

- 如果App交互手势与浏览器原生触摸操作产生冲突，屏蔽原生触摸操作
- 注意优化性能，预览时不要使用性能或者内存开销过高的算法，最后保存图片时再以浏览器提供的最高画质拼接生成完整的长图。
- 基于以下要点对metadata进行SEO优化：“支持纵向/横向拼接长图，支持手动调整接缝以及整体裁切，完全基于浏览器前端，PWA可离线运行”
- 对metadata以及界面进行英文本地化，根据浏览器语言自动切换中文/英文
- PWA化这个应用，使其可以离线运行。
